Starting this blog has been very fun. It was some time since I wrote pure CSS and HTML, which is apparently something I've missed.

Philosophy

I've had a few things in mind when writing the actual code:

  • Design for mobile devices right after you're done with desktop. It's painful to go back and finish the mobile stuff later.
  • Let CSS be quick and dirty - it's static. If it works, it works. Too much technical debt otherwise!
  • Utilize Jekyll!

In retrospect, I should probably write these posts in Markdown, as code snippets are a pain to write in HTML...
